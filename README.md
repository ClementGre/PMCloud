# PMCloud

An efficient cloud based picture manager that supports automatic grouping of pictures into custom hierarchies, while allowing users to share groups together.

## Setup

Make sure to install the dependencies:

```bash
npm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
npm run dev
```

## Production

Build the application for production:

```bash
npm run build
```

Locally preview production build:

```bash
npm run preview
```
